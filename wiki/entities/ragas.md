# Ragas

## What It Is

Ragas is an open-source framework for evaluating RAG pipeline quality. It provides automated metrics for measuring faithfulness, answer relevance, and context quality.

## Key Details

- Evaluates RAG systems on faithfulness (are answers grounded in retrieved context?)
- Part of the CI pipeline for continuous RAG quality assessment
- Integrates with common RAG frameworks
