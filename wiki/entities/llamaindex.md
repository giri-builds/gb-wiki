# LlamaIndex

## What It Is

LlamaIndex is a data framework for building LLM applications that connect to external data sources. It specializes in ingestion, indexing, and querying of structured and unstructured data for RAG pipelines.

## Key Details

- Focused on data connectors and indexing strategies for RAG
- Provides abstractions for document loading, chunking, and retrieval
- Alternative to [[entities/langchain]] for orchestration, with stronger data-focus

## How It Fits In

- Part of the **orchestration layer** in the [[concepts/ai-engineering-stack]]
- Implements [[concepts/retrieval-augmented-generation]] pipelines
- Complements or competes with [[entities/langchain]] depending on use case
