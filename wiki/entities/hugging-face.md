# Hugging Face

## What It Is

Hugging Face is an AI platform and community that provides open-source models, datasets, and training libraries. Its ecosystem includes Transformers, TRL (for fine-tuning), datasets library, and the Hugging Face Hub for model sharing.

## Key Details

- **TRL** (Transformer Reinforcement Learning): library for SFT and DPO fine-tuning
- **Axolotl**: community fine-tuning tool built on the HF ecosystem
- Hub hosts trending papers and open-source models
- Key resource for continuous learning in AI engineering

## How It Fits In

- Primary ecosystem for [[concepts/fine-tuning]] with open-source models
- Hosts models used with [[entities/ollama]] for local inference
- Important for staying current — "follow AI researchers, reading trending papers on Hugging Face"
