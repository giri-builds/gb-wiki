# ChromaDB

## What It Is

ChromaDB is an open-source embedding database designed for simplicity and developer experience. It runs locally or embedded in applications, making it popular for prototyping and smaller-scale RAG systems.

## Key Details

- Lightweight, easy to set up locally
- Good for development and prototyping RAG systems
- Can be embedded directly in applications

- Part of the **retrieval layer** in the [[concepts/ai-engineering-stack]]
