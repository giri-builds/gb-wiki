# OpenAI

## What It Is

OpenAI is an AI research company that builds the GPT family of large language models. It provides proprietary models via API, and its OpenAI Evals framework is a reference for building evaluation systems.

## Key Details

- Builds the **GPT** model family (GPT-5.4 referenced in sources)
- Provides the OpenAI SDK for API integration
- OpenAI Evals Framework provides registry-based eval patterns

## How It Fits In

- Part of the **models layer** in the [[concepts/ai-engineering-stack]] (proprietary models)
- SDK proficiency alongside [[entities/anthropic]] is a key AI engineer skill
- GPT models used in [[concepts/llm-as-a-judge]] evaluation setups
