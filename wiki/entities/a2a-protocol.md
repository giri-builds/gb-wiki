# A2A Protocol

## What It Is

A2A (Agent-to-Agent) is Google's protocol for enabling communication between AI agents. While [[entities/mcp]] handles agent-to-tool communication, A2A focuses on agent-to-agent coordination and collaboration.

## Key Details

- Focuses on inter-agent communication (agent-to-agent) vs MCP's agent-to-tool pattern
- Enables multi-agent collaboration where specialized agents coordinate tasks

## How It Fits In

- Complementary to [[entities/mcp]] in [[concepts/multi-agent-systems]]
- Part of the emerging agent communication protocol landscape
